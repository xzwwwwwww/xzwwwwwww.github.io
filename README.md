# 我的小站

一个零依赖的静态个人网页：无需安装任何东西，双击 `index.html` 就能在浏览器打开。

## 目录结构

```
my-site/
├── index.html      # 页面结构（首页 / 读书笔记 / 生活记录 / 关于我）
├── css/style.css   # 全部样式
└── js/
    ├── data.js     # ★ 你的内容都写在这里
    └── main.js     # 页面逻辑（一般不用动）
```

## 如何添加一篇读书笔记

打开 `js/data.js`，在 `NOTES` 数组最前面加一项：

```js
{
  title: "《书名》读后感",
  date: "2026-07-18",
  tags: ["文学"],
  excerpt: "一句话摘要，显示在首页卡片上。",
  content: [
    "正文第一段。",
    "正文第二段。"
  ]
}
```

保存后刷新浏览器即可。列表和首页都按日期自动排序。

## 如何添加生活记录

在 `data.js` 的 `MOMENTS` 数组里加一项：

```js
{ date: "2026-07-18", text: "今天做了什么。" }
```

## 中英文切换

导航栏右侧有「中 / EN」切换按钮，界面文案即时切换，选择会保存在浏览器里（`localStorage`）。文案字典在 `js/main.js` 顶部的 `I18N` 对象里，想改双语措辞直接改它。笔记正文保持你写的原文，不参与切换。

## 想改样式？

配色集中在 `css/style.css` 顶部的 `:root` 变量里（背景色、主题色、文字色），改几个值就能换一套皮肤。
