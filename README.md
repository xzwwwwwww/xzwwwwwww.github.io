# 我的小站

一个零依赖的静态个人网页：无需安装任何东西，双击 `index.html` 就能在浏览器打开。

## 目录结构

```
my-site/
├── index.html      # 页面结构（首页 / 读书笔记 / 生活记录 / 关于我 / 留言信箱）
├── css/style.css   # 全部样式
├── SETUP-INBOX.md  # 留言信箱后端（Supabase）配置指南
└── js/
    ├── config.js   # ★ Supabase 密钥（按 SETUP-INBOX.md 填写）
    ├── data.js     # ★ 你的笔记和生活记录都写在这里
    ├── inbox.js    # 留言信箱逻辑（一般不用动）
    ├── game.js     # 黄金矿工小游戏（一般不用动）
    └── main.js     # 页面逻辑（一般不用动）
```

## 如何添加一篇读书笔记

打开 `js/data.js`，在 `NOTES` 数组最前面加一项：

```js
{
  title: "《书名》读后感",
  date: "2026-07-18",
  tags: ["文学"],
  excerpt: "一句话摘要，显示在首页卡片上。",
  content: [
    "正文第一段。",
    "正文第二段。"
  ]
}
```

保存后刷新浏览器即可。列表和首页都按日期自动排序。

## 如何添加生活记录

在 `data.js` 的 `MOMENTS` 数组里加一项：

```js
{ date: "2026-07-18", text: "今天做了什么。" }
```

## 留言信箱

访客可在「留言信箱」页自定义昵称留言，留言先进入待审核状态，不会直接公开。页面底部「站长登录」登录后可审核：通过、回复、隐藏、删除。留言数据存放在 Supabase，首次使用请按 `SETUP-INBOX.md` 配置（约 10 分钟），否则信箱显示“暂未开放”。

## 小游戏

「小游戏」页内置 Canvas 版黄金矿工：点开始 → 点击画布/按空格放钩 → 抓金块、钻石得分，60 秒结算。纯前端实现，无后端依赖。

## 中英文切换

导航栏右侧有「中 / EN」切换按钮，界面文案即时切换，选择会保存在浏览器里（`localStorage`）。文案字典在 `js/main.js` 顶部的 `I18N` 对象里，想改双语措辞直接改它。笔记正文保持你写的原文，不参与切换。

## 想改样式？

整站采用「自嘲熊」插画风：粉橘底 + 芥末黄 + 奶白卡片，首页是纯 CSS 绘制的白熊吃拉面动画。配色集中在 `css/style.css` 顶部的 `:root` 变量里，标题字体为站酷快乐体（Google Fonts，index.html 里引入）。
